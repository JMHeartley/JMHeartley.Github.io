PicsOfUs is a photo gallery only available to invited users. It uses Microsoft Azure's Face API for facial detection and recognition. Faces are used to create family members; fill in details to build and view family trees.

  • Photo gallery - the web application is a photo gallery that is only available to invited users.
  • Face recognition - Microsoft Azure Face API is used to detect and recognize faces in uploaded pics. The API is used to create family members and link existing family members to new pics.
  • Family trees - view family trees constructed from relationship data; clicking on any node in the tree will display the person's details and pics.
  • Search - search for pics by selecting faces. The search results will display the pics that contain the selected faces.
  • Responsive design - the web application is designed to be responsive and work on desktops, tablets, and mobile devices.

PicsOfUs is a web application written in C# throughout its stack.

  • Front-end - .NET 5 Blazor, Microsoft's single-page application technology, uses HTML, CSS, Javascript, and C# to dynamically render web pages to the user.
  • Back-end - the Service/Repository pattern is used to separate business logic from data access.
    • Services - the Service layer is responsible for business logic and communication with external services. Written using C#, the asynchronous Service layer methods communicate with the Repository layer to access data.
    • Repositories - the Repository layer is responsible for data access and communication with the database. The database was designed using a code-first approach with Entity Framework Core 6, combining C#, LINQ, and EF Core 6. The SQL Server database is hosted on Microsoft Azure.
  • Testing - the testing framework used is xUnit, with Moq used for mocking. The testing layer is written in C#, uses EF Core 6 with SQLite for in-memory database access, and contains over 100 unit tests.
  • External Services - Microsoft Azure Face API (facial recognition), Microsoft Azure Computer Vision (to reject adult content), Microsoft Azure Virtual Machine (hosting platform), SendGrid (email service)
