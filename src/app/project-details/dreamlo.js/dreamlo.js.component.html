• dreamlo.js is a library for creating, retreiving, updating, and deleting scores from a dreamlo leaderboard
  • dreamlo UI allows anyone to use dreamlo.js without having any of the implementation logic in place

Check out the documentation on GitHub for a detailed explanation and breakdown.
