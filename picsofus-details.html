PicsOfUs: A family photo gallery

  • Category: Web Application
  • Client: Myself
  • Stack: Blazor (.NET 5.0), ASP.NET Identity, Microsoft Azure, Entity Framework Core 6, SQL, Bootstrap 5
  • Languages: C#, Javascript, HTML, CSS

PicsOfUs is a Blazor web application for cloud hosted private photo collections shared between families. It's integrated with several of Microsoft Azure's services. This was my first start up, which failed due to competition with a loss leader, but has since been released to the public as open-source software.
