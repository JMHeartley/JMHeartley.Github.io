PicsOfUs: A family photo gallery

  • Category: Web Application
  • Client: Myself
  • Stack: Blazor (.NET 5.0), Entity Framework Core 6, SQL, Bootstrap 5
  • Languages: C#, Javascript, HTML, CSS
  • Testing: C#, xUnit, Moq, SQLite
  • Libraries: dTree (family tree visualization), Bootstrap 5 (UI), ASP.NET Identity (account management), SharpZipLib (extracting uploads into zip folder)
  • Services: Azure Face API (facial recognition), Azure Computer Vision (to reject adult content), Azure Virtual Machine (hosting platform), SendGrid (email service)

PicsOfUs is a place where your uploaded pics are private and accessible only to invited members.

Have you ever been at Thanksgiving with everyone taking pictures on their phone so everyone can have a picture? With PicsOfUs you can just use the best camera then upload the group picture to the collection.

  • Employs the Service/Repository design pattern to separate business and data access logic
  • Integrates editor for creating/updating family member details from faces of uploaded pictures
  • Facilitates querying of uploaded pictures based on the photo subject(s)
  • Gathers related members, then using recursion, constructs a hierarchical data structure to pass to dTree, an open-source family tree visualization library
  • Enables family tree exploration; any tree node can be clicked to view member details
  • Supported by ~100 unit tests
